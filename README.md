# Orkesta
Linux üzerinde web geliştirme servislerini kolayca kurup yönetebileceğiniz modern, modüler servis orkestrasyon paneli.

## Özellikler
- Apache & Nginx sunucu yönetimi
- MySQL, PostgreSQL ve MongoDB veritabanı kontrolü
- Servis başlat/durdur/yükle/kaldır işlemleri
- Otomatik log görüntüleme ve durum izleme
- Hafif, hızlı ve kullanıcı dostu arayüz (GTK)
- Profil tabanlı stack kurulumu

## Hedef Kullanıcılar
- Web geliştiriciler
- Backend mühendisleri
- DevOps ve sistem yöneticileri
- Laravel, Django, Flask, Node.js, PHP geliştiricileri

## Platformlar
- Linux (Debian, Ubuntu, Arch, Fedora vb.)
